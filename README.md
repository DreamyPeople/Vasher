## Installing Tools
```
pkg update && pkg upgrade
pkg install python git
cd $HOME
git clone https://github.com/DreamyPeople/Vasher
```

## Running
```
cd $HOME/Vasher
python run.py
```

## Updating
```
cd $HOME/Vasher
git pull
```
